AI-Passcode Secure Password Management System

📌 Project Overview

AI-Passcode Secure Password Management System is a secure web application designed to help users store, manage, and protect their passwords efficiently. The application provides secure authentication, password management features, and a modern user-friendly interface.

 ⚙️ Installation

# Clone Repository

```bash
git clone https://github.com/Tejas0116/ai-passcode-secure-password-manager.git
```

# Install Frontend

```bash
cd client
npm install
npm run dev
```

# Install Backend

```bash
cd server
npm install
npm start
```

# 🔐 Environment Variables

Create a `.env` file inside the server folder:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
